# RougePuzzle

## ゲームの概要

|内容    |詳細           |
|:----------|-------------:|
|プレイ人数　|           一人|
|カテゴリー　|　  ローグライト|
|ターゲット  |        自分たち|
|対象OS     | MAC|

## ゲームルール

- １階層での移動回数の制限がある
- １階層に１部屋
- 部屋はn*m
- 部屋内部に階段、壁、プレイヤー、エネミー、アイテムが設置される
- 階層を移動すると移動回数が再設定される
- プレイヤーは階層内で移動回数がゼロになったらゲームオーバー
- アイテム
  - スコアアイテム
  - HP回復アイテム、行動回数回復アイテム
  - その他アイテム 
- 最大スコアを目指す
